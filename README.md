# Sprint 6 - E-Commerce Store

## Project Overview

This project is a React.js based E-Commerce Store developed as part of Sprint 6.

It demonstrates:

- React Router DOM
- Dynamic Routing
- Fetch API
- Context API
- Shopping Cart
- Local Storage
- Responsive UI

---

## Features

- Home Page
- Shop Page
- Product Details Page
- Add to Cart
- Cart Page
- Dynamic Cart Badge
- Local Storage Persistence

---

## Tech Stack

- React.js
- Vite
- React Router DOM
- Context API
- CSS

---

## API Used

https://dummyjson.com/products

---

## Installation

```bash
npm install
npm run dev
```
